Smiley Separation Model

This is an improvement to our smiley separation which now takes the 3 turns as separate inputs into the LSTM layers.

Using TensorFlow backend.

Config

This model was produced with the following configs:

Embedding Dim = 200
Batch Size = 32
LSTM Dim = 128
Dropout = 0
Learning Rate = 0
Num Epochs = 100
Early Stopping = True

Model Specification

An extra hidden layer which is supposed to extract the important information from the

first embedding then add information from smilys
